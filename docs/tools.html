A list of some of the things I use or have used in developing code. Some of which I'm quite passionate about.

Development Environments

  • I use Atom for web-development. I've used Eclipse for Java, and at work I use Netbeans (especially its Swing builder). I've used a smattering of other IDEs in the past (e.g. Komodo for Python development; meh).
  • I'm fond of IDEs in general and love using and customizing them, but I won't be able to let my sense of pride rest until I learn to use Emacs.

OS

  • Windows on my PC, Linux at work. I intend to buy a new computer and install Ubuntu sometime in the near future...
  • Mac? What's that?

Versioning

  • Git and Github for my own projects; SVN at work.

Literature

  • Effective Java by Joshua Bloch is a treasure. My coworkers gave me a copy before I knew any Java and I found it completely bewildering; now I have it by my side whenever I code. Probably the most valuable thing I've gotten at work, aside from experience...
  • The Pragmatic Programmer by Andrew Hunt and David Thomas. Another fantastic resource. I was lucky enough to come across this book right at the beginning of learning to code, so it was able to really shape how I approached development.
